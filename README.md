# 🔍 Portal de Pesquisas Cadastrais

Uma aplicação web desenvolvida para centralizar consultas cadastrais de empresas brasileiras em uma interface moderna, rápida e intuitiva.

O sistema permite realizar pesquisas de informações como **CNPJ**, **Inscrição Estadual** e **Simples Nacional**, facilitando o acesso aos dados em um único ambiente.

---

## 🚀 Tecnologias Utilizadas

* **React**
* **TypeScript**
* **TanStack Router**
* **TanStack Query**
* **Vite**
* **Tailwind CSS**
* **Shadcn UI**
* **Lucide React**

---

## ✨ Funcionalidades

* Consulta de empresas por **CNPJ**
* Consulta de **Inscrição Estadual**
* Verificação de enquadramento no **Simples Nacional**
* Interface moderna e responsiva
* Navegação rápida utilizando TanStack Router
* Gerenciamento eficiente de requisições com TanStack Query

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

Entre na pasta:

```bash
cd seu-repositorio
```

Instale as dependências:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

A aplicação ficará disponível em:

```text
http://localhost:5173
```

---

## 📦 Build de Produção

```bash
npm run build
```

Para visualizar a versão de produção localmente:

```bash
npm run preview
```
